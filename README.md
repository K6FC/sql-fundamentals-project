# SQL-APRENDIZAGEM-FUNDAMENTOS

Repositório da minha primeira semana de estudos em SQL básico, focado no aprendizado de conceitos, funções e operadores, utilizando um dataset de vendas para responder queries 

## OBJETIVO
- Aprender os fundamentos do SQL
- Praticar consultas com uma base de dados real
- SELECT, WHERE, ORDER BY, LIMIT
- Tipos de dados
- Operadores lógicos (AND, OR, IN, BETWEEN)

## Tecnologias usadas
- SQL
- DuckDB
- DBeaver
- Git & GitHub

## Estrutura do Repositório
├── data/Datasets utilizados
├── queries/Consultas SQL
├── notes/Apuntes teóricos
└── README.md

Está sendo utilizado um dataset público do Kaggle, todo com fins educativos.
